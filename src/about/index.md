# 关于 Pallas-Bot

**Pallas-Bot**（群里常叫牛牛）是面向 QQ 群的学习型机器人：从聊天里学复读，附带喝酒、轮盘、决斗等玩法，也可作 **MAA** 的 QQ 侧远控入口。

## 特点

- 学习型复读，跨群语料与「不可以」全局禁用
- 群聊玩法：喝酒、轮盘、决斗、做梦、唱歌、画画、夺舍
- 管理：黑名单、入群/好友申请、帮助图开关
- 运维：Web 控制台、协议端多账号、官方插件按需安装
- 数据：MongoDB 或 PostgreSQL

## 技术栈

NoneBot2 · OneBot v11 · Python 3.12+ · MongoDB / PostgreSQL
