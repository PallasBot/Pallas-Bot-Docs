# 使用指南

面向**部署者与群管理员**：跑起牛牛、在网页里改配置、常用口令与排障。开发见 [开发指南](/develop/index)。

## 推荐路径

1. **[五分钟跑起来](/guide/quickstart)** — 克隆、改 `pallas.toml`、启动、连 QQ
2. 浏览器打开 **`http://<主机>:<端口>/pallas/`**（默认端口 8088）
3. 群里发 **`牛牛帮助`** 看功能列表

不理解「协议端 / 数据库 / 控制台」分工时，可看 [理解架构（可跳过）](/guide/concepts)。

## 网页控制台

| 想做什么 | 去这里 |
| --- | --- |
| 改画画 / 唱歌 / 远控地址 | 通用配置 → **外部服务地址与连通检测** |
| 改智能对话 | 通用配置 → **智能对话与 AI 服务** |
| 改口令权限 / 冷却 | **命令权限** / **命令冷却** |
| 社区共享接话 | **社区共享接话库** |
| 多牛同群不抢话 | **多机协同** |
| 装官方扩展 | 插件 → **官方扩展** |

配置落盘 `data/pallas_config/webui.json`，见 [配置存储](/architecture/settings-storage)。

## 常用口令

以群内 **`牛牛帮助`** 为准。示例：

| 口令 | 作用 |
| --- | --- |
| 牛牛帮助 | 功能总览 |
| 牛牛连通 | 检测外部服务 |
| 牛牛喝酒 / 醒一醒 | 饮酒状态 |
| 牛牛轮盘 | 轮盘玩法 |
| 牛牛 | 打招呼（全员同响时多牛齐回） |

完整索引：[plugins/README.md](/plugins/index)。

## 排障

| 现象 | 先看 |
| --- | --- |
| 没反应 | [FAQ](/deploy/faq) — 拉黑、冷却、插件已关 |
| 智能对话不可用 | 控制台 AI 地址与开关；`牛牛连通` |
| 多牛只回一只 | **多机协同** / 全员同响 |
| 升级旧版 | [3.0 迁移](/about/migration) |
