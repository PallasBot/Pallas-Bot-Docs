# Web 控制台

Pallas Web 控制台提供可视化管理界面。

## 访问地址

- 地址：`http://<HOST>:<PORT>/pallas/`
- 登录口令：首次启动时在 Bot 日志中打印

## 功能

- 插件管理：启用/禁用插件
- 配置管理：编辑插件配置
- 数据库管理：查看/修改数据
- 版本更新：在线更新 Bot

## 运维入口

| 入口 | URL 示例 |
| --- | --- |
| Web 控制台 | `http://<HOST>:<PORT>/pallas/` |
| 协议端管理 | `http://<HOST>:<PORT>/protocol/console/` |
