# repeater（牛牛复读）

群内学习对话、按相似度与重复度自动回复与复读，定时主动发言，管理员可禁止指定内容。

## 行为概要

- 学习群消息并持久化
- 相同消息连续出现达到阈值时复读
- 管理员可通过「不可以」禁止该内容
- 表情回应：受 `enable_reaction` 等开关控制

## 配置

调参见插件配置（如 `answer_threshold`、`repeat_threshold`、`speak_threshold` 等）。

## 排障

| 现象 | 说明 |
|------|------|
| 从不说话或话太多 | 调整阈值配置 |
| 复读不触发 | 检查 `repeat_threshold` |
