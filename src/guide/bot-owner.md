# 号主

号主是**某一只牛**的运维人：写在该牛数据库 `admins` 里的 QQ。与群管理员（QQ 群职位）、超管（全站 `superusers`）不同。

::: tip
连上 QQ 后尽快配好号主。未配置时，私聊「重新上号」、好友欢迎等号主向能力不可用。
:::

## 三种角色

| 角色 | 是谁 | 管什么 |
| --- | --- | --- |
| **超管** | `config/pallas.toml` 的 `[bootstrap] superusers` | 所有牛、控制台最高权限 |
| **号主** | 该牛 `bot_config.admins` 中的 QQ | 这只牛的号主向口令与部分群管级能力 |
| **群管** | QQ 群管理员 / 群主 | 本群内的群管级口令（如帮助里开关插件） |

帮助图「何人可用」以当前生效的 [命令权限](/maintainer/operate/command-permissions) 为准；代码里号主对应等级名多为 `bot_moderator` / `staff`。

## 号主常做什么

| 能力 | 说明 |
| --- | --- |
| 私聊 `牛牛重新上号` | 拉起/重登协议端（需协议端插件） |
| 好友欢迎语 | 设置 / 清除（`greeting`） |
| 私聊状态类口令 | 如部分「在吗」类（以帮助为准） |
| 与群管重叠的能力 | 拉黑、禁用复读语料、「不可以」等（`staff` 级） |

完整口令以群里 **牛牛帮助** 为准。

## 怎么配置

`admins` 为 **QQ 号组成的数组**（整数）。不能写在遗留 `.env` 里。保存后通常很快生效；未生效可稍等或重启 Bot。

### 方式一：控制台「实例与连接」（推荐）

1. 打开 `/pallas/` → **实例与连接**
2. 在 **NoneBot 框架** 面板找到目标牛
3. **库配置** 为未入库时：点 **初始化配置**，填写号主 QQ 并保存
4. 已在库中：在 **数据库中的实例** 编辑号主

不依赖协议端或 `relogin_bot` 是否已装。

### 方式二：控制台「数据库管理」

1. 打开 **数据库管理**
2. 表类型选 `config (bot_config)`
3. 找到或新建 `account` = 牛 QQ 的行，编辑 `admins`（如 `[123456789]`）并保存

### 方式三：超管私聊「牛牛添加号主」

由本体 **`pb_core`** 提供，**仅超管**、须 **私聊** 牛发送：

| 场景 | 示例 |
| --- | --- |
| 为当前私聊的牛添加 | `牛牛添加号主 2777777777` |
| 一次添加多个 | `牛牛添加号主 2777777777 2666666666` |
| 指定目标牛 | `牛牛添加号主 2777777777 牛 3888888888` |

也可 **@ 号主** 代替手写 QQ。目标牛未入库时会自动创建 `bot_config` 并写入；已在库则合并追加。详见 [pb_core](/plugins/pb_core)。

### 方式四：创建牛牛时写入

已装 **`relogin_bot`**（`pallas-plugin-protocol`）时，超管私聊 **创建牛牛** 可带号主 QQ；协议端创建并启动后自动写入 `admins`。账号已从别处连上时，改用方式一或三。
