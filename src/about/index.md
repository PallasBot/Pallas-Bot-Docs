# 关于

Pallas-Bot（牛牛）是一个面向群聊场景的学习型机器人，也可作为 **MAA（MaaAssistantArknights）** 的 QQ 侧远控。

## 项目特点

- 学习型复读，不依赖硬编码问答库
- 支持跨群语料聚合与全局禁用
- 丰富的群聊玩法：喝酒、轮盘、决斗、做梦、唱歌、聊天、生图、夺舍
- 管理能力强：黑名单、好友欢迎、好友/入群申请管理
- 数据后端：MongoDB 或 PostgreSQL
- 运维：Web 控制台、协议端管理
- **MAA 远控**：绑定设备后用口令排队任务，回传截图与状态

## 技术栈

- NoneBot2
- OneBot v11
- MongoDB / PostgreSQL
- Python 3.12+
- MAA 远程控制协议
