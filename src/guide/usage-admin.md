# 日常管理

::: tip
Bot 已启动且能打开 `/pallas/`。尚未连 QQ 见 [连接 QQ](/guide/connect-qq)。
:::

## 登录控制台

1. 打开 `http://<主机>:8088/pallas/`（端口以 `pallas.toml` 的 `[bootstrap] port` 为准）
2. 输入启动日志中的控制台密码
3. 群里发 `牛牛帮助`，对照帮助图确认功能可用

面板说明 → [网页控制台](/guide/web-console)

## 控制台常用操作

| 任务 | 操作 |
| --- | --- |
| 配置从哪改 | 见 [配置从哪改](config.md) |
| 配置号主 | **实例与连接** → 初始化 / 编辑 `admins`；详见 [号主](bot-owner.md) |
| 安装决斗、MAA 等 | **插件商店** → 安装 → 重启 Bot |
| 改画画 / 唱歌 / AI 配置 | **通用配置** |
| 改命令权限 | **命令权限** |
| 关闭某牛全部玩法 | **插件** → **全实例禁用** |
| 排查无响应 | **运行日志**、**日志报错** |
| 添加 NapCat 账号 | **协议端实例** 或 `/pallas/protocol` |
| 备份数据库 | **数据库** |

保存后写入 `data/pallas_config/webui.json`。

## 群里常用口令

完整列表以 **牛牛帮助** 为准。摘要见 [命令与功能](/guide/usage)。

| 口令 | 作用 |
| --- | --- |
| `牛牛帮助` | 功能总览 |
| `牛牛连通` | 测试外部服务（含 AI） |
| `牛牛喝酒` / `醒一醒` | 喝酒状态 |
| `牛牛轮盘` | 轮盘玩法 |

## 超管运维口令

本体自带 **`pb_core`**，默认加载，**仅超管**。

| 口令 | 作用 |
| --- | --- |
| `牛牛状态` / `牛牛插件` / `牛牛控制台` / `牛牛更新` / `牛牛重启` | 进程摘要、插件列表、控制台入口、更新检查、重启 |
| `牛牛添加号主 …` | 私聊为指定牛写入号主；见 [号主](bot-owner.md) |

完整说明 → [牛牛核心 pb_core](/plugins/pb_core)

## 常见问题

| 现象 | 排查 |
| --- | --- |
| 群里完全没反应 | 协议端是否在线；牛是否在群；[FAQ](/deploy/faq) |
| 某功能不可用 | 插件是否安装；帮助图里是否被关掉 |
| AI 不可用 | 控制台 AI 地址；发 `牛牛连通` 测试 |
| 忘记控制台密码 | [FAQ](/deploy/faq) |
