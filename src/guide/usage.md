# 口令与功能

群里发 **牛牛帮助** 可看当前群已启用插件与口令（图里会标「何人可用」）。

::: tip 权限怎么分
**群管** — QQ 群管理员；**号主** — 该牛 `admins` 里的 QQ；**超管** — `pallas.toml` 的 `superusers`。配置号主见 [FAQ](/deploy/faq)。
:::

## 核心玩法

| 口令 / 能力 | 一句话 |
| --- | --- |
| 牛牛帮助 | 三级帮助图、群管/号主可关插件 |
| 复读 / 欢迎 / 夺舍 | 学习型复读、入群欢迎、自动改群名片（被动） |
| 牛牛喝酒 · 醒一醒 | 醉酒状态，影响聊天、轮盘等 |
| 牛牛轮盘 | 禁言轮盘，`牛牛救一下` / `牛牛补一枪` |
| 牛牛做梦 | 语料与图片漂流 |

## AI 扩展（需 Pallas-Bot-AI）

在控制台配置 AI 服务地址；可用 **牛牛连通** 自检。

| 口令 | 说明 |
| --- | --- |
| 随时闲聊 等 | 智能对话（见控制台 LLM 开关） |
| 牛牛唱歌 / 点歌 | AI 翻唱与网易云 |
| 牛牛画画 | 文生图 / 改图 |

详见 [AI 扩展](/guide/ai)。

## MAA 远控

绑定设备后群聊或私聊发令（完整表见 [maa](/plugins/maa)）：

`牛牛绑定MAA` · `牛牛长草` · `牛牛作战` · `牛牛公招` · `牛牛基建` · `牛牛截图` · `牛牛停止`

## 运维口令与页面

| 入口 | 说明 |
| --- | --- |
| `/pallas/` | Web 控制台 — 配置、日志、备份 |
| `/protocol/console/` | 协议端 — NapCat / SnowLuma |
| 牛牛连通 | 探测画画 / MAA / 唱歌网关 |
| 牛牛在吗 / 报数 | 在线状态（[bot_status](/plugins/bot_status)） |
| 重新上号 / 创建牛牛 | 号主 / 超管 |

申请审批、拉黑、拦截其它牛见 [request_handler](/plugins/request_handler)、[blacklist](/plugins/blacklist)、[block](/plugins/block)。
